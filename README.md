# Sietch
Un visualizador de comunidades seleccionadas a tu preferencia que utiliza un algoritmo propio de tendencias para organizar posts en un layout Bento Grid dinámico

## BentoReddit Custom Feed

Este proyecto es un dashboard inteligente que no solo muestra contenido de Reddit, sino que aplica una **fórmula de tendencia personalizada** sobre un conjunto específico de comunidades (subreddits) elegidas por el usuario. El resultado se visualiza en un **Bento Grid** moderno y responsivo en el cual los elementos más grandes serán los de mayor viralidad o tendencia en el cual tambien podras crear grupos de comunidades para adaptar tu feed por categorias propias.
